# 📊 Vendor Performance Analysis

> An end-to-end data analytics project that analyzes vendor performance using Python and Power BI to generate actionable business insights, evaluate supplier efficiency, and support data-driven procurement decisions.

---

## 📖 Project Overview

This project focuses on analyzing vendor sales, purchases, inventory, and profitability to evaluate supplier performance. Using data cleaning, exploratory data analysis (EDA), KPI analysis, and interactive visualizations, the project uncovers trends and provides insights to optimize procurement and improve business performance.

---

## 📂 Dataset

The analysis is based on multiple business datasets containing:

- Vendor Information
- Purchase Records
- Sales Transactions
- Inventory Data
- Product Details
- Pricing & Profit Metrics

The datasets were cleaned, merged, transformed, and prepared for analysis to create a unified, analytics-ready dataset.

---

## 🎯 Business Objectives

- Identify top-performing vendors
- Analyze sales and profit trends

## 📊 Power BI Dashboard

The interactive Power BI dashboard provides a comprehensive view of vendor performance through dynamic visualizations and KPIs, enabling users to:

- Monitor vendor sales and profitability
- Compare vendor performance
- Track procurement trends
- Analyze inventory metrics
- Explore business insights using interactive filters and slicers

---

## 💡 Key Insights

- Identified top-performing vendors based on revenue and profitability.
- Detected vendors with high procurement costs but low profit margins.
- Analyzed purchasing and sales trends across vendors.
- Evaluated inventory utilization and stock movement.
- Compared vendor contributions to overall business revenue.
- Generated recommendations to improve procurement efficiency and profitability.
