# 📊 Vendor Performance Analysis

> An end-to-end Exploratory Data Analysis (EDA) project that evaluates vendor performance using Python to generate business insights and support data-driven procurement decisions.

---

## 📖 Project Overview

This project analyzes vendor sales, purchases, inventory, and profitability data to evaluate supplier performance and identify opportunities to optimize procurement strategies. The analysis includes data cleaning, preprocessing, exploratory data analysis (EDA), KPI generation, and data visualization.

---

## 📂 Dataset

The analysis is based on multiple business datasets containing:

- Vendor Information
- Purchase Records
- Sales Transactions
- Inventory Data
- Product Details
- Pricing & Profit Metrics

These datasets were merged and transformed into a clean, analysis-ready dataset for generating business insights.

---

## 🎯 Business Objectives

- Identify top-performing vendors
- Analyze sales and profit trends
- Evaluate procurement efficiency
- Monitor inventory performance
- Generate actionable business insights
